# ModaBox - E-commerce Store

A modern e-commerce store built with Next.js, React, and Tailwind CSS.

## Features

- Product catalog with categories (Clothes, Shoes, Accessories)
- Product image slider
- Admin panel for managing products
- Product visibility toggle
- Search and filter functionality
- Responsive design

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app router pages
- `components/` - React components
- `context/` - React context for state management
- `lib/` - Data and utilities

## Pages

- `/` - Home page with all products
- `/clothes` - Clothes category page
- `/shoes` - Shoes category page
- `/accessories` - Accessories category page
- `/admin` - Admin panel for managing products
